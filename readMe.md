## Información de interés:

- Juego por turnos con temática 'South Park'.

- Seis personajes seleccionables.

- Peleas de uno contra uno.

- 4 pantallas o fases: 

        FASE 0: INICIO.
        FASE 1: SELECCION DE PERSONAJES.
        FASE 2: PELEA.
        FASE 3: GANADOR.
